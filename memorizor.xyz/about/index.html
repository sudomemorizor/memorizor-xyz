About Me

My name is Memorizor. I also go by "Zero" or "Null" among those i know personally.

I am an ever-learning coder, artist, game dev, web dev, musician, among other things. I created this website to showcase my projects and present myself more professionally, and to learn more about web development.

My resources are somewhat limited, so unfortunately this is all for now. Some day I hope to create web apps and games, and make interactive user experiences.

Art

I've been drawing since I was really small. Drawing is probably the one skill I have practiced the longest. I draw pretty much every day, but most of my work is kept private.

I will occassionally post work I find presentable to the internet, which you can view on my many scattered social media accounts. I plan to create a gallery page with a comment section for my website at some point so you don't have to go digging for it if you'd like to see it all in one place.

Music

Music is a more recent hobby of mine. I have always liked listening to music, but had always wished I could make it myself. Well, I decided "better late than never" and got into music production more seriously. Output is slow, as expected when you have a dozen different hobbies and only so much time to practice them.

You can listen to my music on Soundcloud or Bandcamp
